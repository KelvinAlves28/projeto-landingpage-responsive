Conheça o Databricks

Databricks é uma plataforma de análise e de processamento escalável baseada no Apache Spark - um dos mais populares e poderosos frameworks de processamento de big data e análise de dados, que possibilita fluxos de trabalho simplificados e um workspace interativo. Isso permite a colaboração entre os cientistas de dados, os engenheiros de dados e os analistas de negócios, tudo isso integrado com a cloud para fornecer instalação rápida e segura.

A tecnologia possibilita escrever em qualquer linguagem, como R, Python, Scala e SQL.

Diferenciais

1.Análise exploratória

O Databricks permite que os analistas de dados possam explorar grandes conjuntos de dados, aplicando análises e visualizações com bibliotecas já instaladas, combinando, por exemplo, SQL e Python para explorar dados.

2. Machine learning

O Databricks oferece suporte para construção de modelos de machine learning em grande escala, de forma colaborativa entre equipes, permitindo cuidar de todo ciclo do ML, desde o treinamento até a implantação do produto final.

3. Processamento de dados em lote ou em tempo real

Você pode processar os dados, e montar todo ciclo de ETL, fazendo análises e processamento de fluxos de dados de forma contínua (Streaming) ou em lote.
